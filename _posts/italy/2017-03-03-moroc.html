第二天: 卡萨布兰卡-拉巴特–丹吉尔

早餐：酒店内享用

早餐后，我们将前往拉巴特，参观拉巴特皇宫，乌达雅花园城堡，穆罕默德五世陵墓，哈桑塔。

拉巴特是现代化并保留摩洛哥的特色的城市，有许多迷人的历史遗迹，包括风景如画的Kasbah of the Udayas（乌达雅堡）。联合国教科文组织世界遗产名录上的kasbah(伊斯兰堡垒)建于Almohads的统治时期，建于12世纪中叶;独特的哈桑塔，开始于12世纪末，旨在拥有世界上最大的尖塔，但从未完成。在塔的对面，伫立着穆罕默德五世陵墓，这个陵墓埋葬着摩洛哥国王穆罕默德五世和他的两个儿子的陵墓。您将会发现，这座建筑被认为是现代阿拉尤王朝建筑的杰作。

{% assign imgn = imgn | plus: 1 %}

在拉巴特古老的麦地那，被古老的城墙被围着的城市充满迷宫般的街道，喷泉，宫殿，清真寺和市场。在这里，你会看到Mechouar的城墙，一个被花园包围的、严格保护、极大的皇宫。

{% assign imgn = imgn | plus: 1 %}

早餐：酒店内享用

下午我们将前往天空之城-蓝白小镇舍夫沙万-令人窒息的蓝。

{% assign imgn = imgn | plus: 1 %}

舍夫沙万建立于1741年，一直是游客们最喜爱的旅游目的地，它又被称为北部的蓝色城市，因为城市所有建筑的墙壁涂满了的蓝色和白色油彩，非常静谧而安逸。

{% assign imgn = imgn | plus: 1 %}

舍夫沙万的手工艺非常发达，除了卖一些特色服装、小物件，还有各种各样的编织毯。

第五天: 菲斯

早餐后，全天在非斯市内游览。非斯圣城是西班牙- 阿拉伯文明的宝石。

菲斯是一个非常神秘的城市，它需要被一点一点地发现，只有这样才能充分欣赏 Medrsa建筑的辉煌;也只有这样，麦地那老城的召唤才能吸引到你:熙熙攘攘的工匠和商人，其迷人的声音，香味和颜色吸引了游客参与其中。Moulay Idriss 1st 城市，人类和道德科学圣所。伟大的Karaouiyne大学是当时一个宗教和智力 中心，其影响力远远超出摩洛哥王国的范畴。公元789年，一个由金银合金制作的 镐 - “fas”(阿拉伯语) - 交给伊德里斯一世用于彰显皇城的声望，城市的名字

由此而来。菲斯位于Saïss平原的远东端，与南部毗邻阿特拉斯山脉的山丘，宗 教，文化和艺术传统的摇篮。

{% assign imgn = imgn | plus: 1 %}

午餐：当地特色餐

菲斯是王国的精神首都。在Karaouiyne清真寺周围 的Medersas是专门为来自城外的学生准备的租住房屋，向游客展示了其千年历史 的重要性。这里还有著名的Kissaria商业中心。 许多本地生产的商品在这个令人 难以置信的迷宫里出售，如棉织物，丝绸，锦缎工作，拖鞋。

{% assign imgn = imgn | plus: 1 %}

第六天：菲斯-伊芙兰-米德勒特-埃尔福德-梅祖卡

早餐：酒店内享用

上午出发前往撒哈拉沙漠，沿途拍摄中部阿特拉斯山区优美的自然风光，中途拍摄富有欧风小镇伊芙兰Ifrane（65公里/1小时30分钟），在这里您不会看到任何摩洛哥文化的影子。伊芙兰位于阿特拉斯山区海拔1664公尺，菲斯南方60公里，于1929年开始建城，人口数仅约一万人，每年都吸引无数游客到此度假。

{% assign imgn = imgn | plus: 1 %}

午餐：当地特色餐（自理）

下午驱车前往梅祖卡，途径伊拉琪迪Errachidia，沙漠之门埃尔福德Erfoud和里撒尼Rissani，可以停留片刻游览拍照。

{% assign imgn = imgn | plus: 1 %}

傍晚到达撒哈拉沙漠的腹地梅尔祖卡，可选择骑骆驼（自费）爬到山丘上，漫游于黄金色的沙漠上并同时欣赏广阔撒哈拉沙漠日落。撒哈拉的骆驼都是大型号的，在骆驼背上视野极佳，东边一团流沙，西边一湾沙脊起伏有致的曲线，阳光在上面划来划去，骆驼一个踉跄，惊出你一身冷汗。你的思绪回到三毛随笔中的描述：“由我住的地方到小镇上快要四十分钟，没有车，只好走路去。漫漫的黄沙，无边而庞大的天空下，只有我们两个渺小的身影在走着，四周寂寥得很，沙漠，在这个时候真是美丽极了”。

{% assign imgn = imgn | plus: 1 %}

晚上入住沙漠豪华帐篷，围着篝火，吃着烤肉，欣赏绝美清透的星空。

第八天:达德斯山谷-玫瑰谷-埃本哈度-瓦尔扎扎特-马拉喀什

早餐：酒店内享用。

早餐后，我们将前往女人天堂-玫瑰谷，参与一年一度的“玫瑰节”。每年五月的“玫瑰节”期间，整个山谷从拂晓开始就飘满了玫瑰的芬芳，犹如来自天堂的味道。

{% assign imgn = imgn | plus: 1 %}

在这年度的盛典上，当地的柏柏尔族人们会跳起热情奔放的民族舞蹈，而摩洛哥当地的人民，比世界各地慕名而去的人们更为一年一度的“玫瑰节”兴奋。

{% assign imgn = imgn | plus: 1 %}

在玫瑰谷可尝试海娜(Henna)纹身，在身体、手脚画满图案，象征高贵和富足，向真主、神祈求幸福。所用的颜料来自一种叫“海娜”的植物，通常它可以存在几天或者几个星期之久。

{% assign imgn = imgn | plus: 1 %}

第九天：马拉喀什

【热气球，自费】

早早起床，专车抵达您下榻的宾馆，接上您抵达热气球出发点。早点是有特色的当地油酥面点，咖啡和茶。工作人员在您早餐时准备热气球。

进入热气球的乘座篮，开始热气球之旅。热气球翱翔在沙漠之上，您可以充分享受这里的景色。40-50分钟（依据风向，天气情况）后热气球降落。

早餐：我们回到帐篷里，面朝阿特拉斯山享受早餐。有美味的烙饼，牛角面包，水果，橄榄和薄荷茶。

早餐后骑骆驼，环绕营地，穿过沙漠里的村庄。

颁发热气球飞行证书。

{% assign imgn = imgn | plus: 1 %}

接下来，我们将在马拉喀什游玩一整天，我们将参观著名的历史文物——梅纳拉花园，有着被花包围的壮丽水池和一栋建于1866，为当时贵族所修建的，漂亮的摩式建筑。我们可以欣赏在阿特拉斯山的映衬下壮丽的日落。

{% assign imgn = imgn | plus: 1 %}

接下来我们还将游览几个令人印象深刻的风光：库图比亚宣礼塔（其姐妹塔为位于西班牙塞维利亚的吉拉达塔），彰显摩式完美的Saadian墓群。Saadian墓葬群由建于16世纪华丽精致的墓葬组成，曾经在很长一段时间被围墙围住，直到1917年才被发现。其他区域包括展示柏柏人艺术品的Dar Si Said博物馆;Badii Palace（巴伊亚皇宫），在这个历史纪念馆每年六月的第二个的星期五举行年度流行艺术节。Bahia Palace 仍旧在mallah(犹太人区)招待贵族。

{% assign imgn = imgn | plus: 1 %}

午餐：当地特色餐

参观完这些历史文化遗迹之后，我们将漫步于马拉喀什的大街小巷，伴随着在日落，游览令人流连忘返的Jamaa El Fna Square（杰马夫纳广场，也称不眠广场）。杰马夫纳广场是马拉喀什的灵魂所在，是摩洛哥最大的柏柏尔人露天市场，也是全球为数不多被列入世界文化遗产的广场，这里是马拉喀什人生活的缩影，百年生生不息的各色表演贩卖，兜售工艺品的小贩、商人像开挂似的不断切换各国语言，奇装异服的艺人用表演吸引游客的围观，在这能看到马拉喀什文化的千姿百态，默默诉说着马拉喀什千百年的繁华。

{% assign imgn = imgn | plus: 1 %} {% assign imgn = imgn | plus: 1 %}
