梵高

你就像黑夜，拥有着寂寞和群星  

就像我喜欢的你是寂静的

 

直到大约27岁时，梵高才开始了他的画家生涯，而他最著名的作品多半是他生前最后两年在普罗旺斯小镇阿尔勒和圣雷米所创作的，期间梵高深陷于精神疾病之中，直到37岁那年用自杀的方式结束了自己的生命...

 

南法梵高路线一日游推荐：

  Marseille（马赛）/ Aix-en-provence（埃克斯）/ Avignon (阿维尼翁）→ Arles (阿尔勒)→Saint-Rémy-de-Provence (圣雷米) →Les Baux de Provence (莱博小镇) → Marseille（马赛）/ Aix-en-provence（埃克斯）/Avignon（阿维尼翁）

 

『阿尔勒 Arles』

       1888年，一心想要去燃烧着炽热太阳的城市的梵高来到了阿尔勒，驻足在罗纳河畔边，挥笔画出了色彩艳丽，充满着痛苦与希望的《罗纳河上的星夜》。那样漆黑的夜空，在他眼中竟然绚烂如斯。而后这幅画也是著名民谣歌手Don McLean创作《Vincent》的这首歌的灵感来源。

{% assign imgn = imgn | plus: 1 %}

阿尔勒是法国普罗旺斯一座小城，游走在充满着艺术文化气息的文化古城里，不仅可以感受到古罗马露天剧场的魅力，更让人难以忘怀的天才画家梵高曾在此留下的印记。在梵高生命的最后两年，他在这座城挥洒了全部的热情，他的疯狂、他的执念、他的孤傲，他对美好生命的呐喊...

{% assign imgn = imgn | plus: 1 %}

与画家高更，两人因艺术问题外加梵高身体病痛，经常发生争吵。一次争吵之后，高更愤怒之下离他而去。梵高因情绪激动而导致精神失常，于12月23日晚挥刀割掉自己的左耳。

{% assign imgn = imgn | plus: 1 %}

这是一位生活在伤口里的大师，他习惯用伤口对世界发言。这是一个疼痛的收割者，他的镰刀最终收获了自己的耳朵。

{% assign imgn = imgn | plus: 1 %}

梵高医院（Espace Van Gogh）这里就是梵高割掉自己的耳朵以后被送进去疗养的医院，所以在当地还有一个名字就是疯人院，因为梵高被当时小镇的人看作是个疯子，他们甚至要求市政厅把梵高驱逐出这个城市。

{% assign imgn = imgn | plus: 1 %}

“向日葵，那一定是属于我的花。”梵高这样对弟弟提奥说，那样金色的，炙热如太阳般的颜色，是画家的最爱，是飞向太阳的鸟，热爱会让身躯融化掉。他的向日葵就像一个个动人的不停跳跃的音符，传递着强烈的生命欲望，也许梵高追随自己内心选择来到阿尔就好像向日葵的花语一样-----勇敢的去追寻自己想要的幸福。

{% assign imgn = imgn | plus: 1 %}

梵高在1888年完成的《夜间的露天咖啡座》被认为达到了他的艺术成就颠峰。是当代文化最著名的代表之一，也是梵高的作品中最著名的一个。它也是被复制最多，印刷最多的油画。

{% assign imgn = imgn | plus: 1 %}

在梵高完成此画的100多年后, 这家咖啡馆一直是阿尔勒的人们饮酒的地方，现在被称做梵高的咖啡馆，仍旧保留着那黄色的雨蓬和所有的东西。也有很多游客因为那幅画而专门到此喝上一杯，感受一下画中情景。当然，咖啡馆中已找不到当年梵高用以消愁的苦艾酒，替代它们的是啤酒、葡萄酒和白兰地。

{% assign imgn = imgn | plus: 1 %}

『圣雷米 Saint-Rémy-de-Provence 』

      梵高的油画《圣雷米疗养院与教堂风景》于2012年2月7日在伦敦拍卖出了1600万美元的新高价。而创作出这幅画的梵高在当时却穷困潦倒。如果当时的梵高能得到这样一笔巨款的话，一定会改变他的人生轨迹，然而，没有如果。

{% assign imgn = imgn | plus: 1 %}

这时的梵高已陷入对精神病的恐惧和对前途的迷茫中。他每隔几个月发一次病，但事后却非常清醒，并且经常到户外作画。令人惊叹的是，梵高在这样的情形下并没有颓废，开始以火一般的热情继续画画。

{% assign imgn = imgn | plus: 1 %} {% assign imgn = imgn | plus: 1 %}

此时，他画的不是阿尔城的向日葵画，而是柏树和橄榄树画，画出了一幅幅更加成熟，更加大胆，更加令人震撼的作品。疗养院里有一个花园，那里也是梵高经常作画的地方。在梵高精神状况好的情况下，梵高还被允许到疗养院的附近散步和作画，所以那里很多地方都成了梵高的画中之景。

{% assign imgn = imgn | plus: 1 %}

在圣雷米疗养院期间，梵高一共完成了150多幅油画和100多幅素描。其中最著名的代表作就是《星夜》 ，竟和一位韩国摄影师曝光星空的景象高度相似。天才与疯子往往只是一步之差，这也许就是梵高眼里的世界。

『莱博小镇  Les Baux de Provence』 

      莱博小镇（Les Baux de Provence）是但丁地狱中世纪堡垒位于阿尔比勒（Alpilles）山脉核心区域，它海拔245米，也是法国著名的山顶小镇。历史上最早占领此地的是利古里亚人，公元前五世纪这支哥特分支将此地命名为:“Bau”，即陡峭的岩石，小镇的法文名字意思即为：普罗旺斯的陡峭岩石。

{% assign imgn = imgn | plus: 1 %} {% assign imgn = imgn | plus: 1 %}

莱博被米其林旅游指南列为最高的三星级必游的旅游景点，和其他相似的法国山顶小镇相比，莱博有与之相同的山顶村落风光，更有与之不同的军事要塞堡垒特色。残存的古堡和旧房屋构成了莱博村庄(Village des Baux)独有的山区风景线, 彰显了古城昔日的辉煌。莱博村庄在山顶已有几百年历史。漫步其中，仿佛有能感受到中世纪城堡的繁荣景象。特别的是，法国人喜欢来这里度假，这里几乎还看不见亚洲人面孔。

{% assign imgn = imgn | plus: 1 %} {% assign imgn = imgn | plus: 1 %}

光窖（影像大教堂）Carrières de Lumières 位于地狱谷近旁，在历史上是地下采石场。现在这里被改造成了先进前卫的光艺术场所，超过6000平方米的墙体被投影设备照亮，形成壮观美丽的图案。墙上、脚下、眼中，到处都是流动的美景，伴随着光与影的变化再加上石洞内自然的环绕立体声效，简直就是一场视觉和听觉的盛宴！！
